- **URL**: `/movies/discover/:page`

### Find Trending Movies

Fetch currently trending movies.

- **URL**: `/movies/trending/:page`
- **Method**: GET
- **URL Parameters**:
  - `page` (required): The page number.
- **Response**: Returns a JSON object with an array of trending movies ( just the same as `discover` ) .
